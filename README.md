# DataAnalystProject1
Buisness Problem: Frequent delivery delays were causing penalty losses and poor cuctomer satisfaction with no clear visibility into root causes across regions,couriers and warehouses.

My Solution: Built a Logistics Performance Analytics Dashboard using Python(Pandas) and Power BI to monitor delivery delays, courier efficiency, warehouse performance and penalty losses.
This Dashboard solving Problem by highlight these points-
1-The region which has highest delay rate
2-The courier partner which causing most delay
3-How much penalty loss is happening
4-The order which are in high risk
5-The warehouse which is underperforming
6-The combination of region+courier which is worst
7-Average Delay Days per region
8-The courier constributes most to penalty loss
9-The day which has most delays

Buisness Impact: Enabled data driven actions that improved on time delivery performance reduced penalty losses and optimized logistics operations.
