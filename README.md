# DataAnalystProject1
Buisness Problem: Frequent delivery delays were causing penalty losses and poor cuctomer satisfaction with no clear visibility into root causes across regions,couriers and warehouses.

My Solution: Built a Logistics Performance Analytics Dashboard using Python(Pandas) and Power BI to monitor delivery delays, courier efficiency, warehouse performance and penalty losses.
This Dashboard solving Problem by highlight these points-

1-The region which has highest delay rate (region_delay_rate1.csv)

2-The courier partner which causing most delay (courier_delay_rate.csv)

3-How much penalty loss is happening (most_penalty.csv)

4-The order which are in high risk (high_risk_orders.csv)

5-The warehouse which is underperforming (warehouse_performance.csv)

6-The combination of region+courier which is worst (combination_performance.csv)

7-Average Delay Days per region (average_delay_per_region.csv)

8-The courier constributes most to penalty loss (penalty_table.csv)

9-The day which has most delays (day_order_delay.csv)

Buisness Impact: Enabled data driven actions that improved on time delivery performance reduced penalty losses and optimized logistics operations.
